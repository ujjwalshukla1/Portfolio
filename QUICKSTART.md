# ⚡ Quick Start Guide

Get your portfolio running in 5 minutes!

## 1️⃣ Install Dependencies
```bash
npm install
```

## 2️⃣ Set Up Sanity
```bash
npx sanity@latest init
```
Answer the prompts to create your Sanity project. Your `.env.local` will be auto-configured.

## 3️⃣ Deploy Schema
```bash
npx sanity@latest deploy
```

## 4️⃣ Start Dev Server
```bash
npm run dev
```
Visit `http://localhost:3000`

## 5️⃣ Add Content
```bash
npx sanity@latest desk
```
Or visit: `https://your-project-id.sanity.studio`

Create documents:
- **Hero** (1) - Your name, title, description
- **About** (1) - Your story
- **Skills** (5-10) - Your skills by category
- **Projects** (3+) - Your best work
- **Contact** (1) - Your contact info

## Done! 🎉

Your portfolio is now live with:
- ✨ GSAP animations
- 🎨 Modern dark design
- 📱 Responsive layout
- 🚀 Fast performance
- 🎛️ Easy content management
